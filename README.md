<h2>📖 Description</h2>
<p>
  This project is a lightweight <b>Alarm Clock and Timer</b> application written in Python. 
  It allows users to set specific countdowns using a <code>minutes:seconds</code> format. 
  Whether you need a quick pomodoro timer or a simple reminder, this script handles the timing 
  precisely and alerts you when the time is up.
</p>

<h2>🚀 Features</h2>
<ul>
  <li><b>Customizable Timing:</b> Input time in a flexible MM:SS format.</li>
  <li><b>Live Countdown:</b> Real-time display of remaining time in the terminal.</li>
  <li><b>Audio/Visual Alert:</b> Notifies the user immediately once the countdown reaches zero.</li>
  <li><b>User-Friendly Interface:</b> Clean and simple command-line prompts.</li>
</ul>

<h2>🛠️ Installation & Usage</h2>

<h3>Prerequisites</h3>
<p>Ensure you have Python installed on your machine. You can check by running:</p>
<pre><code>python --version</code></pre>

<h3>Setup</h3>
<ol>
  <li>Clone this repository:
    <pre><code>git clone https://github.com/yourusername/alarm-clock.git</code></pre>
  </li>
  <li>Navigate to the project directory:
    <pre><code>cd alarm-clock</code></pre>
  </li>
  <li>Run the application:
    <pre><code>python alarm_clock.py</code></pre>
  </li>
</ol>

<h2>🖥️ How It Works</h2>
<p>
  Once the script is running, simply enter your desired time. The logic follows a standard countdown loop:
</p>
<blockquote>
  <b>Input:</b> 02:30 <br />
  <b>Action:</b> The script converts this to 150 seconds and counts down to zero.
</blockquote>
